MBUTY is for reading / recording / analyzing the VMM3A, CAEN R5560 and SKADI data saved in pcap or from kakfa 

--------------------------------------------------

libReader for CAEN R5560, SKADI and VMM
Plots and analysis for VMM MG and MB detectors 

--------------------------------------------------

the repository is cloned via git as git clone https://gitlab.esss.lu.se/detectorgroup/multiblade/mbuty.git

MBUTY.py is the backend and runs from spyder or terminal, settings are at the bottom of the file. 
MBUTY_GUI.py is the GUI and can be started from terminal as: >> pyhton MBUTY_GUI.py 

For installing the GUI with a desktop ICON in Linux/Mac OS or Windows execute in a terminal:
>> python install_gui.py
